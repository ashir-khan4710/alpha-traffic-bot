## 🔧 Customization Guide

### Change Colors
Edit in `frontend/index.css`:
```css
:root {
    --accent-orange: #FF8A00;      /* Change orange */
    --accent-purple: #8B5CF6;      /* Change purple */
    --accent-green: #00D26A;       /* Change green */
    --bg-primary: #070B14;         /* Change bg */
}
```

### Change Sidebar Width
```css
.sidebar {
    width: 280px;  /* Adjust to your preference */
}
```

### Add Glow Effect
```css
box-shadow: 0 0 20px rgba(255, 138, 0, 0.3);  /* Orange glow */
```

### Modify Menu Items
Edit in `frontend/index.html`:
```html
<a href="#new" class="nav-item" data-section="new">
    <span class="icon">🎨</span>
    <span class="label">New Section</span>
</a>
```
